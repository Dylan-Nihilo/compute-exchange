# Compute Exchange Domain

本上下文统一算力撮合、Token 服务、居间线索和信任存证使用的业务语言，避免账户、身份、资金与额度等概念混用。

## 账户与参与方

**账户（Account）**：
一个自然人或组织登录平台的统一主体，可在资质通过后拥有多个身份。
_Avoid_：用户角色、工作台

**身份（Identity）**：
账户经认证后获得的业务参与资格，包括买家、供给方、设备厂商和资方。
_Avoid_：账户类型

**买家（Buyer）**：
购买算力商品或 Token 套餐的已认证身份。
_Avoid_：客户账号

**供给方（Supplier）**：
提交机房资质并通过强审核后，可发布算力商品和履约的身份。
_Avoid_：卖家、机房账号

**设备厂商（Vendor）**：
发布设备或服务并承接居间线索的企业身份。
_Avoid_：供给方

**资方（Funder）**：
接收分配的融资租赁线索并登记线下跟进结果的企业身份。
_Avoid_：放款方账号

## 交易与履约

**算力商品（Compute Product）**：
供给方按 GPU 规格、地域、计费方式、可售时段和余量挂牌的算力供给。
_Avoid_：算力实例、机器

**算力订单（Compute Order）**：
买家对一个算力商品发起并直接支付的交易承诺，承载锁量、履约、退款和风控状态。
_Avoid_：交易、合约

**交付凭证（Fulfillment Credential）**：
供给方回填并向买家脱敏展示的算力开通证明。
_Avoid_：算力密码、实例本身

**分账记录（Split Record）**：
订单款项中供给方应得、平台佣金和通道费的持牌支付侧处理记录。
_Avoid_：平台余额、平台清算

## Token 服务

**Token 套餐（Token Package）**：
买家购买的模型调用服务商品。
_Avoid_：充值商品

**Token 额度（Token Quota）**：
Token 套餐兑现到模型网关的可消费服务配额，不具有现金价值。
_Avoid_：Token 余额、账户余额、储值

## 审核、风控与信任

**资质（Qualification）**：
身份或供给能力必须提交并经平台审核的证明集合。
_Avoid_：KYC 状态

**合规开关（Compliance Gate）**：
控制平台处于仅展示、限额真实交易或全量真实交易状态的全局业务门槛；限额模式允许撮合与支付，但受单笔和累计额度限制。
_Avoid_：功能开关、支付开关

**风险告警（Risk Alert）**：
由规则或监测信号产生、需要运营处置的可疑行为记录。
_Avoid_：违规结论

**存证（Attestation）**：
对已确认业务事实计算指纹并异步写入区块链后形成的可验证记录。
_Avoid_：链上订单、原始数据上链

**居间线索（Lead）**：
设备、施工或融资需求进入 CRM 后的分配与跟进记录。
_Avoid_：线上融资订单
