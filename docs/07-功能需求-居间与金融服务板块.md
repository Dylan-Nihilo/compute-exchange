# 07 · 功能需求 · 居间与金融服务板块

> 需求编号：`REQ-E-xxx`。
> 涵盖业务板块 3、4 的居间部分 + 板块（设备融资租赁）。决策 D4：**融资租赁仅展示+留资，不做系统化撮合**。

## 7.1 板块构成

| 子板块 | 形态 | v1 实现 |
|--------|------|---------|
| 设备销售居间（服务器/GPU/机电/UPS/制冷） | 撮合线索 + CRM + 佣金台账 | ✅ |
| 机房机电/制冷/机柜施工总包居间 | 项目居间接单 + 进度管理 + 佣金 | ✅（轻量项目管理） |
| 设备融资租赁居间 | **仅展示 + 需求留资（邮件/联系方式）** | ✅（留资，不做撮合系统） |
| 硬件进销存 | 进销存台账 | ◐ P1（基础台账，非完整 ERP） |

## 7.2 设备销售居间

- **REQ-E-001 (P0)** 居间需求登记：需求方（机房/客户）填写设备清单、规格、数量、预算、地域、交付期。
- **REQ-E-002 (P0)** 线索进入 CRM 线索池：状态机 `新建→待分配→跟进中→已报价→已成交/已关闭`。
- **REQ-E-003 (P0)** 线索分配：运营分配给设备厂商(vendor)或自营跟进；厂商在工作台认领。
- **REQ-E-004 (P0)** 报价与成交登记：厂商/运营登记报价、成交金额、佣金率。
- **REQ-E-005 (P0)** 佣金台账：居间成交佣金计算、结算状态、导出。
- **REQ-E-006 (P1)** 设备库（vendor 发布的设备）展示与检索，需求方可直接询价转线索。

## 7.3 机电/施工总包居间（轻量项目管理）

- **REQ-E-010 (P0)** 施工居间需求登记：项目类型（机电/制冷/机柜/总包）、规模、地点、预算、工期。
- **REQ-E-011 (P0)** 项目线索 CRM：同 7.2 线索流转。
- **REQ-E-012 (P1)** 项目进度管理：里程碑/节点、附件（合同/图纸）、进度更新、回款节点登记。
- **REQ-E-013 (P1)** 居间佣金/服务费台账。

## 7.4 设备融资租赁（展示 + 留资）

- **REQ-E-020 (P0)** 融资租赁介绍页：业务说明、合作资方（可匿名/品牌墙）、适用场景、流程图、**邮件 + 联系方式 CTA**。
- **REQ-E-021 (P0)** 需求留资表单：主体名称、设备金额、期望期限、联系人、电话/邮箱、备注；提交进 CRM。
- **REQ-E-022 (P0)** 线索分配给资方(funder)/运营线下跟进（与 docs/06 资方工作台联动）。
- **REQ-E-023 (P0)** **合规声明位**：明确"融资租赁由持牌资方提供，平台仅居间撮合，不从事放贷/吸储"（规避金融牌照风险，见第11章）。
- **REQ-E-024 (P2)** 在线方案匹配/合同/放款——明确不做（后续阶段，需牌照）。

## 7.5 硬件进销存（P1 基础台账）

- **REQ-E-030 (P1)** 设备进销存台账：采购入库、销售出库、库存、成本/售价、毛利。
- **REQ-E-031 (P1)** 与居间订单/佣金台账打通。
- **REQ-E-032 (P2)** 供应商管理。

## 7.6 内容与展示

- **REQ-E-040 (P0)** 五大板块门户展示页（CMS 可编辑）：算力撮合、Token 工厂、设备居间、机电施工、融资租赁。
- **REQ-E-041 (P0)** 案例/资质/合作伙伴展示（CMS）。
- **REQ-E-042 (P1)** 询价/留资统一入口与转化追踪。

## 7.7 验收要点
1. 设备/施工居间：登记→分配→报价→成交→佣金台账 可走通。
2. 融资租赁：展示页 + 留资 → 进 CRM → 分配资方，含合规声明。
3. 所有留资线索可在运营 CRM 统一查看跟进。
