# 算力撮合交易平台 + AI Token 工厂

轻资产撮合平台：不自建机房、不买 GPU。把全国合规机房的闲置/在租算力挂牌撮合交易，配套 AI Token 代销（Token 工厂）、设备/机电居间、融资租赁留资，以及区块链信任存证（机房信誉/履约记录上链，不可篡改、第三方可验）。

**平台只赚撮合、差价、佣金和服务费，不碰重资产。**

## 需求文档

- `需求方案v1.1（区块链+算力）.pdf` — 整合汇报版（业务全景、第一版范围、核心流程）
- `docs/01-14` — 分模块开发细节文档（功能清单）

## 关键约束（红线，永远不做）

1. 虚拟货币"挖矿" — 系统层面结构性排除 + 自动识别 + 强制冻结
2. 资金"二清" — 钱只走易宝持牌账户，平台不碰资金，自动分账
3. 算力理财/份额认购/拉人返利 — 产品形态彻底不做

## 第一版范围（只做网页，手机适配，无 App/小程序）

- 算力在线挂牌撮合交易所（挂牌→下单→在线支付→自动分账→对账）
- AI Token 工厂（独立系统对接、套餐、额度、API Key、账单）
- 区块链信任基础设施（信誉/履约上链存证）
- 四类用户自助入驻：买家 / 机房 / 设备厂商 / 资方，各有工作台
- 设备/机电施工居间（登记 + CRM + 佣金台账）
- 融资租赁：只做展示 + 留资，不做系统
- 运营后台（含"合规开关"：证照办齐前不开真实收款）

第一版不做：算力调度系统、融资租赁放款、自建大模型训练、多语言、跨境结算。
